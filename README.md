# SPICE Mockups

## Project structure

 - `templates/` contains standalone "mockup" html.
 - `partials/` contains reusable components.

## Build

Complile SASS: `sass --watch styles.scss:styles.css`

## Run

Host the root directory with any fileserver, e.g:
```bash
$ npm install -g http-server
$ http-server
```
